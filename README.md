# Practical GRC Risk Identification Workshop

**Hands-on lesson materials for practical Risk Identification** using the Cause → Event → Impact method.

Designed for live GRC streaming (Twitch/YouTube) and real organizational workshops.

## What's Included

| File | Description |
|------|-------------|
| `LESSON_PLAN.md` | **Expanded 2h45m–3h lesson plan** with deep facilitation guidance, input-to-risk extraction exercises, live quality labs (refining weak → strong statements), ownership/accountability discussion, linkages to Vulnerability Management / Change Management / audit evidence, scoring realism, and clear success indicators. |
| `CASE_STUDY.md` | Apex Financial Services – Mobile Banking App Launch + Public Cloud Migration scenario |
| `templates/risk_register_template.csv` | Blank professional Risk Register (ready for Google Sheets / Excel) |
| `templates/sample_populated_risk_register.csv` | Fully worked example with 6 realistic Cause → Event → Impact risks |
| `templates/risk_identification_inputs_checklist.md` | Practical checklist of inputs real organizations actually use |
| `docs/facilitator_guide.md` | OBS setup, screen-share tips, chat engagement prompts, common pitfalls |

## How to Use

1. Clone or download the repository.
2. Import the Risk Register CSV template into Google Sheets or Excel.
3. Follow the detailed Lesson Plan for a live walkthrough or internal workshop.
4. Use the Apex Financial Services Case Study as the common scenario.
5. Encourage chat/audience participation during live risk identification and quality review.

## Key Features of the Expanded Lesson Plan

- Strong emphasis on **real organizational practice** (not theory)
- Structured **live risk-building** block with facilitation techniques
- **Quality Lab** exercises for refining vague/technical statements
- Explicit connections to **Vulnerability Management**, **Change Management**, and **audit expectations**
- Clear **practice challenge** for participants to apply immediately

## Educational Use

These materials are provided for educational and professional development purposes. They represent general best practices and are not a substitute for organization-specific risk management advice, legal advice, or regulatory counsel.
